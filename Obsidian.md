Obsidian은 Markdown 기반의 개인 지식관리 PKM(Personal Knowledge Management) 도구로, 텍스트 메모를 연결하고 시각화 하여, 생각을 구조화 하고 확장할 수 있게 돕는 앱입니다.
GitHub의 README와 같은 Markdown문법을 사용하여 문서를 정리합니다.
## 📝 옵시디언 마크다운 문법 정리
| 문법             | 설명           | 사용 예시           | 출력 결과             |
| -------------- | ------------ | --------------- | ----------------- |
| `# 제목`         | H1 (가장 큰 제목) | `# 프로젝트 개요`     | # 프로젝트 개요         |
| `## 소제목`       | H2           | `## 개발 환경`      | ## 개발 환경          |
| `### 소소제목`     | H3           | `### VSCode 설정` | ### VSCode 설정     |
| `**굵게**`       | 텍스트 강조       | `**중요한 내용**`    | **중요한 내용**        |
| `*기울임*`        | 이탤릭체         | `*참고사항*`        | _참고사항_            |
| `~~취소선~~`      | 취소선          | `~~삭제됨~~`       | ~~삭제됨~~           |
| `---` 또는 `***` | 구분선(수평선)     | `---`           | ――――――――――――――――― |

## ✅ 코드 블록 (Code Block)

 ● 한 줄 코드 (inline)
`` `코드` `` → 예: `python manage.py runserver`

● 여러 줄 코드
<pre> <code> ```python def hello(): print("Hello, world!") ``` </code> </pre>

● 콜 아웃(Callout) 코드 블럭 접기
> [!note] 여기에 내용을 작성
> 여기는 설명을 작성

## ✅ 체크리스트

- [ ] 해야 할 일 
- [x] 완료된 일

단축키 Ctrl + L

---

### ✅ 링크 & 이미지

●  다른 페이지와 다른 페이지 연결
``` markdown
[[다른노드이름-페이지이름]]
```

●  같은 페이지내  제목 및 소제목 연결
```markdown
[[현재페이지 이름# 소제목]]
```

●  다른 페이지내  제목 및 소제목 연결
```markdown
[[다른페이지 이름#소제목]]
```

●  외부링크
```markdown
[Google](https://google.com)
```

● 이미지 삽입
```markdown
[[이미지.png]]
```

## ✅ 목록 (리스트)

● 순서 없는 리스트 (Bullet List)

```markdown
- 항목 하나 
 - 항목 둘  
  - 하위 항목
```

>  `>` 
>  `**중요:**` `-` 또는 `*` 뒤에 **반드시 한 칸 띄기**! 

 



✅ 출력:

- 항목 하나
    
- 항목 둘
    
    - 하위 항목

---

● 순서 있는 리스트 (Numbered List)
```markdown
1. 첫 번째 단계
2. 두 번째 단계
```
✅ 출력:

1. 첫 번째 단계
    
2. 두 번째 단계
